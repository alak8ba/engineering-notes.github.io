    kore-hexagonal • Environnement technique

    Architecture applicative

    Couche Package
    Domaine dev.kore.domain
    Application dev.kore.application
    Infrastructure dev.kore.infrastructure
    Adaptateurs dev.kore.adapter

    Stack backend

    • Java 21
    • Spring Boot 3.2
    • Spring Data JPA
    • Spring Security + JWT
    • Apache Kafka
    • PostgreSQL
    • Micrometer + Prometheus

    Stack frontend

    • React 19
    • TypeScript
    • Vite
    • Tailwind CSS

    Infrastructure locale (Docker Compose)

    services :
  app       # Spring Boot (port 8080)
  postgres  # PostgreSQL 15 (port 5432)
  kafka     # Apache Kafka (port 9092)
  zookeeper # Zookeeper (port 2181)
  grafana   # Grafana (port 3000)
  prometheus # Prometheus (port 9090)

    Profils Spring

    Profil dev

    • PostgreSQL local : localhost:5432, credentials kore/kore
    • Kafka local : localhost:9092
    • Hibernate : update, show-sql: true
    • CORS : http://localhost:5173
    • JWT secret hardcodé pour le développement
    • Logs : DEBUG sur dev.kore, SQL Hibernate visible

    Profil prod

    • PostgreSQL via Docker Compose hostname : postgres:5432
    • Credentials via variables d'environnement
    • Kafka via variable d'environnement
    • Hibernate : validate
    • CORS : domaine de production uniquement
    • JWT secret via variable d'environnement
    • Logs : INFO/WARN, aucun SQL visible

    Endpoints REST

    Méthode Endpoint Description
    POST /api/v1/auth/login Connexion
    POST /api/v1/auth/register Inscription
    POST /api/v1/auth/refresh Renouvellement token
    POST /api/v1/auth/logout Déconnexion
    POST /api/v1/bookings Créer un booking
    PUT /api/v1/bookings/{id} Mettre à jour un booking
    GET /api/v1/bookings/{id} Lire un booking (write side)
    GET /api/v1/projection/bookings Liste paginée (read side)
    GET /api/v1/projection/bookings/{id} Détail (read side)
    GET /actuator/health Santé applicative
    GET /actuator/prometheus Métriques (port 8081)

    Kafka

    Propriété Valeur
    Topic principal kore-booking-events
    Topic DLT kore-booking-events.DLT
    Group ID kore-booking-consumer
    Acks all
    Idempotence producteur true

    Démarrage local

    # Démarrer les dépendances
docker compose -f docker/docker-compose.yml up -d postgres kafka zookeeper

# Lancer le backend
cd backend && ./mvnw spring-boot:run -Dspring-boot.run.profiles=dev

# Lancer le frontend
cd frontend && npm run dev

    Démarrage complet via Docker

    docker compose -f docker/docker-compose.yml up -d
